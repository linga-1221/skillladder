# Skill Ladder — AI-Powered Career Development Platform
🌐 Live Demo: https://skillladder.vercel.app

An intelligent platform helping job seekers and recruiters through 
AI-driven resume evaluation, personalized job recommendations, and 
real-time application tracking.

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: FastAPI, Python
- AI/NLP: spaCy, PyPDF2, pdfplumber
- Database: Firebase Firestore
- Auth: Firebase Authentication

## Key Features
- ATS-based resume evaluation with skill gap analysis
- AI-driven job & learning resource recommendations
- Real-time application tracking dashboard
- Recruiter pipeline management & hiring analytics
- IEEE Published Research Project

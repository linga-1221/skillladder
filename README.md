# AspiroNest

A comprehensive, role-based job application and recruitment management portal.

## Structure
- `backend/` - Python FastAPI backend
- `frontend/` - React.js + Tailwind CSS frontend

## Getting Started

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`
